# Calibration

This document explains how to calibrate the Swan Countdown Timer.

## Calibration goals

- Ensure every digit lands on the correct flap position.
- Remove mechanical backlash or slop in the drive.
- Verify the display reads consistently across all digits.

## Calibration steps

1. Home each digit module.
2. Cycle through every flap position.
3. Confirm the physical stop and sensor alignment.
4. Adjust motor steps, end stops, or detents as needed.

## Validation

- Check numerals and symbols for consistent front-facing alignment.
- Verify the same physical position for repeated cycles.
- Log any mechanical or firmware offset corrections.
