# Printing

This document records the 3D printing strategy for the Swan Countdown Timer.

## Pilot strategy

- Print a small batch of 50 flaps first.
- Validate hinge motion, part fit, and finish.
- Adjust the design before printing the rest of the flaps.

## Printer settings

Suggested starting settings for Bambu A1:

- Layer height: 0.20 mm
- Walls: 3
- Infill: 15% gyroid
- Supports: none if possible
- Brim: off unless lifting occurs
- Material: matte PLA

## Quality checks

- First layer adhesion
- No elephant foot
- Consistent wall thickness
- Clean hinge geometry

## Organisation

- Keep prints sorted by symbol and batch.
- Use labeled trays or bags for easy assembly.
