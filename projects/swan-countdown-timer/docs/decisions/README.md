## How to add a decision

1. Create a new markdown file with a descriptive name, e.g. `2026-07-09-flap-symbol-set.md`.
2. Use the template below.
3. Link the decision from the relevant docs page.

## Decision template

```md
# Decision: <short title>

- **Date:** YYYY-MM-DD
- **Status:** Proposed / Approved / Rejected / Superseded
- **Context:** What problem or choice is being addressed.
- **Options considered:**
  - Option A
  - Option B
  - Option C
- **Decision:** Which option was chosen.
- **Rationale:** Why this option was selected.
- **Consequences:** What follow-up actions are needed.
```
