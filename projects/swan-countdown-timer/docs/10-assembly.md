# Assembly

This document describes the build and assembly process for the Swan Countdown Timer.

## Assembly flow

1. Assemble one digit module frame.
2. Install the wheel and drive mechanism.
3. Fit the pilot flaps and verify movement.
4. Connect the electronics and test actuation.
5. Repeat for each additional digit module.

## Assembly tips

- Work with one digit at a time.
- Use temporary guides or fixtures to align the wheel.
- Verify that each flap can be inserted and removed cleanly.
- Keep fasteners and connectors organized.

## What to verify

- Wheel rotation is smooth.
- Flaps engage and release correctly.
- Each digit homes reliably.
- Wiring is neat and serviceable.
