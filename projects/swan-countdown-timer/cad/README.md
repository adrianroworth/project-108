CAD and 3D modeling files for the Swan Countdown Timer.

## Workflow intent

- Keep editable design-source files in `source/`.
- Keep exported print artifacts in `stl/` and related print-output folders.
- When Blender is used for part authoring or mesh preparation, keep the Blender source and exported print artifacts synchronized.

For detailed policy and workflow, see `../docs/18-blender-and-print-file-workflow.md`.
