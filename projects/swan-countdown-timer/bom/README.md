# Swan Countdown Timer Bill of Materials

This folder uses a staged BOM format so purchases and planned items stay organized as the build evolves.

## BOM structure

- master-bom.md: master ledger of all purchased and planned parts across all stages.
- stage-01-first-digit-prototype.md: items for the current first-digit prototype phase.
- stage-02-post-prototype-expansion.md: items intentionally deferred until after the first digit works.
- stage-03-full-display-build.md: placeholder list for full timer rollout needs.

## How to maintain this BOM

1. Add every new part to master-bom.md.
2. Also add it to the relevant stage file.
3. Update status as Planned, Purchased, Tested, or Replaced.
4. Keep links, ASIN, quantity, and approximate price in sync.

## Current status summary

- Active phase: first digit split-flap prototype.
- Several core prototype components are already purchased.
- Audio, RTC, permanent soldered electronics, and cosmetic lighting remain intentionally deferred.
