# Hieroglyph Artwork

This folder stores themed glyph artwork used on split-flap symbols.

## Expected files

- `h-01.svg` through `h-16.svg`
- Optional alternates: `h-<id>-alt.svg`

## Design rules

- Keep all glyphs centered and optically balanced.
- Match visual weight to numeral glyphs so flips feel consistent.
- Follow safe-area and scaling rules from `docs/13-lettering-and-flap-graphics.md`.

## Notes

- Symbol IDs must match `docs/03-symbol-inventory.md`.
- If symbol count or IDs change, update docs before generating production flap files.
- Explicit symbol register and asset-tracking table: `symbol-register.md`.
